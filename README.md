## ğŸš€ Quick Start

### Prerequisites

- **Node.js** 18.x or higher
- **npm**, **yarn**, or **pnpm** package manager
- **OpenAI API Key** (for AI features)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/mrbrightsides/sentinel.git
cd sentinel
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. **Configure API Keys** (Optional - hardcoded for demo)

The demo comes with API keys pre-configured. For production:
- Edit `src/app/api/ai/*/route.ts` files
- Replace with your OpenAI API key
- Configure other service endpoints as needed

4. **Run development server**
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

5. **Open in browser**
```
http://localhost:3000
```

### Build for Production

```bash
npm run build
npm start
```
